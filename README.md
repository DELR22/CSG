# Concept Canvas

A single-page, self-contained web app for interior design students to move from user research to a fully articulated Value Proposition.

Built for studio teaching at **The University of Alabama**.

## What it does

Guides students through seven structured steps:

| Section | Stage | Task |
|---------|-------|------|
| A | Research | Define user profiles |
| B | Empathy | Map what each user would and would not want |
| Synthesis | Define | Identify common elements; describe users; set conceptual intent |
| C | Frame | Write How Might We questions |
| D & E | Ideate | Generate conceptual directions and design strategies |
| F | Specify | Write the Core Design Solution |
| G | Reflect | Articulate the Value Proposition |

## Deployment

Standalone HTML — no build step, no dependencies, no server required.  
Hosted via **GitHub Pages** at the root `index.html`.

All fonts are loaded from Google Fonts. The app works offline once fonts are cached.

## Export

Students can export a complete `.txt` brief covering all seven sections, or copy the Value Proposition and Core Design Solution to the clipboard.
